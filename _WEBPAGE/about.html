    • About this object

    This is a 3D object created with Tilt Brush.

    It was created as part of a school assignment.

    The object portraits an abstract 3D clock, with 12 sections arranged in a spiral order.

    As a piece of abstract art, its meaning differs between viewers, who are, for example, feeling either depressed or joyful.
